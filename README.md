## 📌 О проекте

Данный репозиторий содержит исходный код **персонального лендинга** для профессионального копирайтера, редактора и автора текстов **Бориса Очкивского**.

Сайт разработан с целью презентации профессиональных услуг, демонстрации портфолио и привлечения клиентов. Дизайн выполнен в светлых, чистых тонах с акцентным золотым цветом, что подчеркивает статусность и доверие. 

**🔗 Ссылка на действующий сайт:** [https://borisochkivskii.github.io/boris-landing/](https://borisochkivskii.github.io/boris-landing/)

---

## ✨ Функционал и разделы лендинга

Сайт адаптирован под мобильные устройства и десктоп. Включает в себя следующие блоки:

*   🖼️ **Промо-баннер:** Привлекающий внимание широкоформатный баннер (в стиле профилей на биржах фриланса) с указанием ключевых компетенций.
*   📝 **Об авторе:** Краткая биография с акцентом на уникальный гибридный профиль (инженерное + психологическое + текстовое образование).
*   🛠️ **Услуги:** Список основных предложений: копирайтинг, редактура, SEO, подбор визуалов, работа с AI.
*   📊 **Навыки:** Интерактивные шкалы-прогрессбары, наглядно демонстрирующие уровень владения инструментами и компетенциями.
*   📂 **Портфолио:** Карточки реальных кейсов с кратким описанием, прямой цитатой и кнопками перехода на полноценные статьи.
*   📞 **Контакты и формы:** Удобные кнопки для перехода в соцсети (VK, Telegram, MAX) и простая форма обратной связи.

## 🛠️ Технологический стек

Проект написан на чистом стеке без использования внешних фреймворков, что обеспечивает его быстродействие:

*   **HTML5** — семантическая верстка;
*   **CSS3** — Flexbox, Grid, кастомные переменные (`:root`), адаптивная верстка (Mobile-first);
*   **Google Fonts** — типографика (шрифты Manrope и Merriweather);
*   **Vanilla JavaScript** — для работы модального окна (Политика конфиденциальности) и плавного скролла.
